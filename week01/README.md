#Week01 第一周作业

#### 1.（选做）自己写一个简单的 Hello.java，里面需要涉及基本类型，四则运行，if 和 for，然后自己分析一下对应的字节码，有问题群里讨论。
编写 Hello.java 和 MathBase.java 文件 使用 ```javac Hello.java MathBase ``` 命令同时编译两个文件，解决依赖问题
使用Idea ```Jclasslib``` 插件 和 ```javap``` 命令查看字节码文件。
